TheMather9000

What is TheMather9000?

TheMather9000 is a tool capable of computing simple calculations, graphing and solving 8 different types of functions by extracting the derivative, calculating the root or evaluating the f(x) value of a given x value.

The Calculator Function

The Calculator Function of TheMather9000 is capable of solving addition, subtraction, multiplication, division, exponentiation and extraction of the square root.

How do I use it?

It is used just like a regular calculator with some minor quirks! I will get into these now, but first I will list some examples.

  • 2+2
  • 2-2
  • 2*2
  • 2/2

These are the regular permissible operations of addition, subtraction, multiplication and division. When it comes to exponentiation or extracting the square root, the unconventional starts!

  • 2**2
  • sqrt(2)

This syntax should be used although a "^" will also work for exponentiation. Brackets are usable just like always.

The Graphing Function

The Graphing Function of TheMather9000 is capable of portraying 8 different kinds of functions graphically, these consist of:

  • Polynomial Functions!
  • Rational Functions!
  • Trigonometric Functions!
  • Exponential Functions!
  • Logarithmic Functions!
  • Piecewise Functions!
  • Absolute Value Functions!
  • Square Root Functions!

All of these are also solvable with the Solving Function.

How do I use it?

The first 5 on the list are entered normally with the aforementioned syntax of operations from the Calculator Function section. I will now give examples on how the other functions should be entered:

  • Piecewise((x, x > 1), (x**2, x <= 1))
  • Abs(x - 3)
  • sqrt(x + 4)

These functions must be entered this way or they will not work.

The Solving Function

The Solving Function of TheMather9000 is capable of solving all of the aforementioned functions. They should be entered in the same manner.

How do I use it?

After entering the function and selecting the function type, an operation should be selected. The derivative, roots and evaluation are options. If the evaluation function was chosen, one must also enter a number at which the function should be evaluated.
